API Endpoints

Authentication

POST /api/auth/register

Register a new user account

POST /api/auth/login

Login with email and password

POST /api/auth/logout

Logout the current user

Users

GET /api/users

Get all users (admin only)

GET /api/users/:id

Get user by ID

PUT /api/users/:id

Update user profile

DELETE /api/users/:id

Delete user account

Events

GET /api/events

Get all events

GET /api/events/:id

POST /api/events

PUT /api/events/:id

DELETE /api/events/:id

Delete event
